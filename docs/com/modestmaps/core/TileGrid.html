maxChildSearchproperty 
public var maxChildSearch:int = 1

if we don't have a tile at currentZoom, onRender will look for tiles up to one levels further in. set this to 0 if you only want the current zoom level's tiles WARNING: bad, bad nasty recursion possibilities really soon if you go much above 1 - it works, but you probably don't want to change this number :)

maxParentLoadproperty 
public var maxParentLoad:int = 0

if maxParentSearch is enabled, setting maxParentLoad to between 1 and maxParentSearch will make requests for lower zoom levels first

maxParentSearchproperty 
public var maxParentSearch:int = 5

if we don't have a tile at currentZoom, onRender will look for tiles up to 5 levels out. set this to 0 if you only want the current zoom level's tiles WARNING: tiles will get scaled up A LOT for this, but maybe it beats blank tiles?
